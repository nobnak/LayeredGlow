# Layer-based Glow effect on Unity

## Installation
Add scoped repository:

- URL: https://package.openupm.com
- Scope: jp.nobnak

Install "[Layered Glow](https://openupm.com/packages/jp.nobnak.layered_glow/)" on Package Manager.
